# Site vitrine — Anthony Renard

Site vitrine personnel présentant mon profil d'**Ingénieur en Intelligence Artificielle** et mon appétence pour la **vulgarisation de l'IA**.

Objectif : sensibiliser le grand public aux bénéfices concrets de l'IA et de l'automatisation (gagner du temps, supprimer les tâches à faible valeur ajoutée, aider à la décision).

## Stack

Site **statique** — aucune dépendance, aucun build :

- `index.html` — structure et contenu (bilingue FR/EN)
- `css/styles.css` — mise en forme
- `js/main.js` — bascule de langue, menu mobile, animations au défilement

## Lancer en local

```bash
python -m http.server 8123
# puis ouvrir http://127.0.0.1:8123
```

Ou ouvrir directement `index.html` dans un navigateur.
